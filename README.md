# Tailwind CSS Template

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/kzamanbd/tailwind-starter
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Preview production build

```bash
npm run start
```
